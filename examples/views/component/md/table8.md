| 属性           | 说明                       | 类型     |        默认值                                          |
|:--------------|:--------------------------|:--------|:-----------------------------------------------------|
| data          | 数据源  | Array  |        []              |
| border          | 是否有外边框和内边框  | Boolean  |       false             |
| trClass          | 设置行的样式，第一个参数为trData，第二个为行index  | Function  |       false             |
| noDataText          | 配置无数据时的提示，未空则不提示  | String  |       ''             |
