| 属性           | 说明                       | 类型     |        默认值                                          |
|:--------------|:--------------------------|:--------|:-----------------------------------------------------|
| value       |  表示当前激活的是那一项         | String\|Number  |        1              |
| type       |  可选值有card, 如果没配置则显示默认样式        | String  |                      |
| closable       |  选项是否可删除       | Boolean  |        false              |
| draggable       |  选项是否可拖拽       | Boolean  |        false              |
