|  属性   |                                 说明                                 |        类型         |       默认值        |
|:------- |:-------------------------------------------------------------------- |:------------------- |:------------------- |
| width | 宽度，当面板是水平方向有效                                      | String、Number |               |
| height | 高度，当面板是垂直方向有效                                      | String、Number |               |
| min | 最小宽度/高度                                      | String、Number |               |
| max | 最大宽度/高度                                        | String、Number |               |
